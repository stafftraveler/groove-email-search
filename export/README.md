# Export Directory

This directory contains all exported contact data from the Groove API.

## Files

- **groove_all_contacts.json** - Your exported contacts in JSON format (auto-generated, not tracked in git)
- **groove_all_contacts.csv** - Your exported contacts in CSV format (auto-generated, not tracked in git)
- **groove_contacts_sample.json** - Sample data (20 fictional contacts) for reference
- **groove_contacts_sample.csv** - Sample data in CSV format for reference

## Note

The `groove_all_contacts.*` files are automatically generated when you run the fetch scripts and are excluded from version control (see `.gitignore`). Only the sample files are tracked in git.
